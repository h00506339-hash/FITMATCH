Getting Started with FitMatch

1. Build Your Wardrobe

Start by uploading clear photos of your clothing items. Organize them by category (tops, bottoms, dresses, etc.) and add colors/styles for better AI matching.

2. Save Inspirations

Upload outfit photos that inspire you from anywhere - Pinterest, Instagram, or your own photos.

3. Analyze with Gemini AI

Click "Analyze with Your Wardrobe" to see how well each inspiration matches your existing clothes using Google Gemini AI. Get a percentage match and see which items you already have.

4. Get AI Suggestions

Receive AI-powered outfit suggestions using only the clothes in your wardrobe. Discover new combinations you never thought of!

🔑 Configuring Google Gemini API

To use real AI features:

  1. Go to console.cloud.google.com
  2. Create a new project
  3. Enable the Generative Language API
  4. Create an API key
  5. Replace the key in the CONFIG section at the top of this file

Pro Tips:

  • • Use clear, well-lit photos for better AI analysis
  • • Add tags to help AI understand style preferences
  • • Upload from multiple angles for key pieces
  • • Check back regularly for new styling ideas
